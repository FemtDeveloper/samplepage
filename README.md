## 🚀 Getting Started


1. Install dependencies
```bash
$ npm Install
```

2. Start dev server

```bash
$ npm run dev
# or
$ yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 🛡 NextAuth.js

You will find the example sign-in page under `/src/pages/auth/signin.js`.
